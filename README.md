 Daytona ABM

Sistema de gestión para clientes, proveedores y productos.

 Funcionalidades

- ABM completo (Alta, Baja, Modificación) de:
  - Clientes
  - Proveedores
  - Productos
- Gestión de registros activos e inactivos
- Filtrado en tablas por nombre, DNI, email, etc.
- Integración con base de datos SQL Server
- Frontend en HTML/CSS/JS puro
- Uso de API pública para cargar provincias y localidades

 Tecnologías

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Base de datos: SQL Server
- Librerías adicionales: Bulma, Bootstrap
